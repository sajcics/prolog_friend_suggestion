Social networking application using deductive database

This project is created for course "Database theory" at University of Zagreb, Faculty of organisation and informatics on last year of Master study.

Description

Project use deductive database to suggest you new friend that are friends of your friend. Someting like Facebook suggestions. When you add person as your new friend, on backend Prolog will called and return new list of suggestions.

Used technologies:

  • Flask framework - Python
  • Jinja 2 - template engine
  • SPADE for XSB
  • deductive database DATALOG
  • HTML
  • CSS
